Queue structure

qhead → [H|•] → [front|•] → [n₂|•] → ··· → [rear|NULL]
                    ↑ Front (dequeue here)                  ↑ Rear (enqueue here)

H = empty header node (dummy, data = 0)
Front = qhead→next, the next node to be dequeued
Rear = pointer to the tail node; new nodes are appended here
FIFO: elements leave from the front, enter from the rear
