Singly Circular — Structure

head → [H|•] → [n₁|•] → [n₂|•] → ··· → [nₖ|•] ↩
                                              ↑
last ──────────────────────────────────────────┘

H = header node (dummy, data=0)
last = pointer to tail node
Last node's next → head (circular)

Doubly Linked — Structure

      ← prev           ← prev
dhead ⇔ [•|n₁|•] ⇔ [•|n₂|•] ⇔ ··· ⇔ dhead
      next →            next →

dhead = header node (circular doubly)
Each node has prev and next pointers
dhead.prev → last node (O(1) tail insert)
